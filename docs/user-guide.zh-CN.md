# 使用指南（审计 Photo Hub）

本指南说明现场用户和管理员的基本操作流程。

## 1. 登录
1. 打开应用 URL。
2. 按 `登录`，通过 Cognito 登录。
3. 首次使用时，输入显示名称。

## 2. 房间

### 创建房间
1. 在 `新建` 中输入房间名称。
2. 按 `创建房间`，或从菜单打开 `房间 > 创建`。
3. 创建后会进入该房间。

注意:
- 每位用户只能创建一个房间。
- 即使已经通过邀请加入其他房间，也可以创建自己的房间。

### 通过邀请加入
1. 打开管理员发送的邀请 URL。
2. 登录后会自动加入邀请目标房间。

### 切换房间
1. 打开画面上方的 `当前房间` 下拉框。
2. 选择要进入的房间。
3. 在确认对话框中按 `OK`。

### 退出房间
1. 打开右上菜单，按 `退出`。
2. 确认警告对话框后执行。

注意:
- 退出后，没有邀请 URL 将无法重新加入该房间。
- 管理员不能退出自己的房间。请从房间管理中删除房间。

## 3. 文件夹和照片

### 创建文件夹
1. 从右上菜单打开 `文件夹 > 创建`。
2. 输入文件夹标题并按 `创建`。
3. 如有需要，可设置文件夹密码。

### 上传照片
1. 从 `当前文件夹` 下拉框选择文件夹。
2. 使用 `上传` 批量添加图片。

### 评论
1. 打开照片卡片中的 `评论`。
2. 输入评论并按 `添加`。
3. 可以编辑或删除自己的评论。管理员可以管理所有评论。

### 导出
1. 从 `当前文件夹` 选择目标文件夹。
2. 从右上菜单打开 `文件夹 > 导出`。
3. 选择导出格式。
4. `PDF`: 查看最稳定。
5. `轻量 PPT`: 体积较小的 PowerPoint 文件。某些手机可能无法显示。
6. `高画质 PPT`: 质量更高的 PowerPoint 文件。某些手机可能无法显示。
7. PDF 导出会在原画面显示进度。
8. 生成的导出文件为临时保存，7 天后自动删除。

推荐顺序:
- 先使用 `PDF`
- 然后使用 `轻量 PPT`
- `高画质 PPT` 适合 PC 确认或正式提交

## 4. 管理员菜单

管理员可在 `房间管理` 中执行:
- 生成或失效邀请 URL
- 设置成员查看范围
- 删除房间和全部数据
- 增加存储容量

## 5. 常见问题

### 房间不显示
- 打开 `当前房间` 下拉框确认列表。
- 如果没有已加入的房间，请向管理员获取邀请 URL。

### 无法上传图片
- 可能因容量不足而暂停上传。请联系管理员。

### 无法登录
- 确认 MFA 验证码是否为最新。
- 如果设备时间不同步，认证容易失败。请启用自动时间设置。
