## 前提
- フロントを開けること
- `localStorage` に以下設定済み
  - `kansa_api_base`
  - `kansa_photo_bucket`
- 試験ユーザーを2名用意
  - ユーザーA（例: tanaka）
  - ユーザーB（例: sato）

## 記録ルール
- 結果: `OK` / `NG` / `保留`
- NG時は「備考」に再現手順とスクリーンショット保存先を書く
